## 1.17.1. Multi-layer Perceptron

`Multi-layer Perceptron (MLP)`은 학습 데이터와 함수 `f: R_m -> R_o`로 학습되는 지도학습 알고리즘이다. 이때 `m`은 차원의 갯수이고 `o`는 출력의 갯수이다. <br>
`X = {x1, x2, ..., xm}` feature들과 `y` 출력이 주어졌을 때, 분류와 회귀 모델에 대하여 비선형 근사치 함수로 학습될 수 있다. <br>
`logistic regression` 모델과 비교하여 입력층과 출력층 사이에 추가적인 비선형층(은닉층이라고 불림)를 추가하는 것이 다르다. <br>
Figure 1은 스칼라 출력을 반환하고 1개의 MLP 은닉층을 보여준다. <br>

![1](/daehyun99/99_images/1.png)

가장 왼쪽의 층(입력층으로 알려진)은 입력 feacture를 나타내며 `x_i = (x1, x2, ..., xm)로 구성된다. <br>
은닉층의 각 뉴런들은 이전 층의 weighted linear summation(`w1x1 + w2x2 + ... wmxm`)을 `tanh`와 같은 비선형 활성화 함수로 변환한다. <br>

이 모듈은 `coefs_`와 `intercepts_` public attributes를 포함한다. <br>
`coefs_`는 가중치 행렬(weight matrice) 목록이다. 인덱스 `i`는 `i`층과 `i+1`층 사이의 가중치 행렬값을 의미한다. <br>
`intercepts_`는 편향 벡터(bias vector) 목록이다. 인덱스 `i`는 `i+1`층에 더해지는 편향값이다.

## 1.17.2. Classification

[`MLPClassifier`](https://scikit-learn.org/stable/modules/generated/sklearn.neural_network.MLPClassifier.html#sklearn.neural_network.MLPClassifier)는 [`Backpropagation`](http://ufldl.stanford.edu/tutorial/supervised/MultiLayerNeuralNetworks/#backpropagation_algorithm)을 사용해서 학습하는 MLP 알고리즘이다. <br>

MLP는 2개의 배열로 학습한다. : <br>
`(n_samples, n_features)` 크기의 배열 X (floating point feature vector로 표현되는 학습 샘플을 들고 있다.) <br>
`(n_samples,)` 크기의 y 배열로 (학습 샘플에 대한 target value(클래스 라벨)을 들고 있다.) <br>

```py
from sklearn.neural_network import MLPClassifier
X = [[0., 0.], [1., 1.]]
y = [0, 1]
clf = MLPClassifier(solver='lbfgs', alpha=1e-5,
                    hidden_layer_sizes=(5, 2), random_state=1)
clf.fit(X, y)
```

학습 후에는, 새로운 샘플에 대한 예측을 수행할 수 있다.

```py
clf.predict([[2., 2.], [-1., -2.]])
# array([1, 0])
```

MLP는 학습 데이터의 비선형 관계도 학습 가능하다. `clf.,coefs_`는 모델의 가중치 행렬값 파라미터들이 들어있다.

```py
[coef.shape for coef in clf.coefs_]
# [(2, 5), (5, 2), (2, 1)]
```

현재, `MLPClassifier`는 오직 `Cross-Entropy loss function`만 지원하며, `predict_proba` 메서드로 확률을 추정할 수 있다.<br>
<br>

MLP는 Backpropagation로 학습한다. <br>
더 정확히 말하자면 gradient descent와 gradients들로 `Backpropagation`을 계산한다. <br>
분류 모델의 경우, 샘플 `x`에 대하여 `P(Y|X)`로 주어진 확률 추정치 벡터로 `Cross-Entropy loss function`값을 최소화한다.

```py
clf.predict_proba([[2., 2.], [1., 2.]])
# array([[1.967e-04, 9.998e-01],
#        [1.967e-04, 9.998e-01]])
```

`MLPClassifier`는 `Softmax` 활성화 함수로 다차원 분류도 지원한다. <br><br>

게다가, 모델은 다중 클래스 분류도 지원한다. <br>
각 클래스에 대하여, 각 행의 출력은 logistic function을 통과한다. <br>
0.5보다 크거나 같은 값은 1로 매핑되고, 이하의 값들은 0으로 매핑된다. <br>
하나의 샘플의 예측된 결과에 대하여, 인덱스 값이 1인 값들은 해당 샘플에 할당된(존재하는) 클래스들이다.

```py
X = [[0., 0.], [1., 1.]]
y = [[0, 1], [1, 1]]
clf = MLPClassifier(solver='lbfgs', alpha=1e-5,
                    hidden_layer_sizes=(15,), random_state=1)
clf.fit(X, y)
# MLPClassifier(alpha=1e-05, hidden_layer_sizes=(15,), random_state=1,
#               solver='lbfgs')
clf.predict([[1., 2.]])
# array([[1, 1]])
clf.predict([[0., 0.]])
# array([[0, 1]])
```

더 많은 정보를 얻기 위해 [`MLPClassifier.fit`](https://scikit-learn.org/stable/modules/generated/sklearn.neural_network.MLPClassifier.html#sklearn.neural_network.MLPClassifier.fit) 문서를 살펴보세요
